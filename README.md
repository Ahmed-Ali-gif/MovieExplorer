## MovieExplorer is a frontend-only project built to practice and strengthen core web development concepts including:

- JavaScript fundamentals
- DOM manipulation
- API integration
- Responsive layouts
- CSS Grid & Flexbox
- Dynamic UI rendering
- Local Storage

**The project focuses on building real-world frontend development confidence by combining HTML, CSS, and JavaScript into a complete application.**

---

## Search Movies

- Search movies dynamically using API requests
- Instant movie results rendering

---

## Movie Cards

Each movie card displays:

- Movie poster
- Movie title
- Release year
- Rating

---

## Favorites System

- Add movies to favorites
- Remove movies from favorites
- Favorites stored using Local Storage

---

## Dark Mode
- Toggle between dark and light themes
- Smooth UI experience

---
